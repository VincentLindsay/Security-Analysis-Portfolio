# Overview
This lab features the use of an Ubuntu server, and a Windows 10 virtual machine to deploy a splunk instance, and gain practical experience building dashboards, reports, and configuring splunk to suit my needs.

Since this lab is based on MYDFIR's Splunk 101 course, a capstone project will entail everything I've learned through the course, and additional documentation (e.g Building the environment) will be added to showcase my learning.
